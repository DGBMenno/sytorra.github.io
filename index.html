About

I’m Menno Weijers. I work on software engineering projects within embedded systems, desktop applications and IoT systems. I like simple, reliable systems with clean interfaces and clear diagnostics.

  • Languages/Tools: C++, C#, Python, HTML, Git
  • Domains: Embedded, tooling, small web apps
  • Approach: small iterations, measurable outcomes, maintainable code

Software Engineering

Development of desktop utilities, C# WinForms/WPF applications, and small web backends. I focus on building integrations, automation scripts, and tools that make workflows faster and more reliable. Clean, maintainable code with a focus on practical delivery.

Embedded Systems

Experience with microcontrollers, low-level coding, and interfacing hardware components. I design and build embedded solutions for real-time control, data collection, and device communication. Reliability and efficiency are always top priority.

IoT Development

Creating IoT solutions that connect devices, sensors, and platforms. From prototyping with Raspberry Pi/Arduino to cloud integration, I work on reliable communication systems (LoRa, MQTT, REST APIs) that bring data online securely.
